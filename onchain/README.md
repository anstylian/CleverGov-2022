# On-chain

## Installation

```bash
npm install
```

## Run tests

npx hardhat test

## Configuration

Create `onchain/.env` based on `onchain/.env.example`
Modify `hardhat.config.js` for blockchain to use

## Run

### Deploy

```bash
npx hardhat run scripts/deploy_all.js
```

### Open new request

```bash
npx hardhat run scripts/open_requests.js
```
